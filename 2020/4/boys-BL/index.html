爺們的BL：閃亮腐海就讓宅宅大叔帶你探索 讀後筆記

Posted on 2020-04-19 Edited on 2022-03-15 In Reading

看到鉛筆與橡皮擦，你會想到什麼

BL腦補的兩個必要條件

  1. 將兩個角色(擬人化的也算)的關係解釋成戀愛關係
  2. 兩者必須都是男性角色

BL腦補的作法

  1. 先對角色性格的解釋進行補充修正 (哪方比較有男子氣概)
  2. 構築兩者之間的關係 (他們如何看待彼此)
    • 以此決定哪一方在精神上佔有優勢
    • 哪一方處於接受的立場
    • 被動的一方是單純的接受，還是想像成有包容力等

BL腦補的開始

  1. 誰在精神上佔優勢
  2. 實在戀愛時佔上風
  3. 誰有哪種自卑情結
  4. 如何【注意】對方

BL & Yaoi 的定義

BL: 描繪男性之間戀愛的原創作品
Yaoi: 將原創作品中的角色關係解讀為戀愛關係的心理活動或創作活動

Yaoi的魅力: 自己從作品中找出萌角
BL則是對象與想像都具體呈現在眼前

解讀

對象與主體的混淆

  1. 作品 - 萌的【對象】發生什麼事
    • 對象上發生什麼事
    • 作品中發生什麼事
  2. 解讀 - 從對象身上產生萌感的【我】發生什麼事

各種讀法

  1. 少女心讀法: 與主角產生共鳴，把自己置換為主角 (登入型)
  2. 腐女子讀法: 想像角色之間不可能發生的關係
  3. BL讀法: 難性之間的戀愛是必要挑建，在作品中已經是戀愛關係了。

角色萌與關係萌的混淆

你喜歡的是

  • 個體/角色本身 OR
  • 兩個人之間的關係

愛角色的方式

  1. 帶入角色(那個角色:成為、被愛，別的角色:去愛他)
  2. 對角色做…，看反應
  3. 純觀察

關係

  • 年齡
  • 外型
  • 社會角色
  • 日常/夜晚

讀法的組合

帶入到 * 想像嗎(BL:不想像,Yaoi:想像) * 主動嗎(純觀察於否)

帶入到

  1. 愛的角色
  2. 對手
  3. 他人
  4. 不到任何人

可逆/不可逆

不可逆: 喜歡的是關係(社會關係)
可逆: 喜歡的是兩人之間的牽絆 <= 純愛主義者

BL => 純愛: 你是唯一

電影評論與Yaoi

以理性的評論最為解釋對象的手法

以感性的行為作出名為想像的創作

人物關係圖

每個人都要有(角色戲份越重要問越多)

  • 【A眼中的B是什麼樣的?A如何看待B?】
  • 【氫元素是這樣的元素，鎳元素是這樣的元素，所以氫元素對鎳元素的看法一定是這樣沒錯】，而非【氫元素對鎳元素有什麼看法】

還有周遭世界的縝密設定

為什麼是男男

少女漫畫一定要

一邊瞄寫男女情感，並且盡力切割性欲要素

喜歡百合的理由

  1. 男性很礙事，一旦是男女關係救回被拉回現實
  2. 描寫男女關係的作品，心中會出現【我有沒女友】，【我有沒辦法像情侶一樣】，像是gal中的選項沒有自己想說的話一樣 => 【我跟他不一樣】

為什麼是同性不可

男女關係中【接受】的一方是確定的

為什麼是男性不可

  1. 可以和【(一部分的)性欲】切割=與【萌】文化的共通處=柏拉圖式戀愛=【純愛物語】
  2. 男人受傷也無所謂
  3. 男人無論精神或肉體都不是【楚楚可憐】的
  4. 可以看到男人【有感覺】的樣子
  5. 可以看到男人苦惱的樣子 (=基本上與【萌】共通)
  6. 享受把男人放進【小小世界】的樂趣(身為支配者的喜悅、【消費】男人、觸動母性…)

因為喜歡你的內在所以愛上你。我其實不也不是不喜歡女人，但還是你比較好

男人受傷也無所謂?

in my opinion, 受傷是可以使用在男人上的證明 (角色為了得到什麼，而要付出的過程)

女性多BL?

  1. 從少女漫畫產生
    • 主角的性別相同(很難不帶入這邊)
    • 到最後女性都是接受的一方
    • 最後產生【我跟他不一樣】
  2. 現實感過強

So, 女性想在BL中得到?

  • 將【憧憬的戀愛】【憧憬的關係】投射其中
  • 想身為第三者，以局外人的身分【在一旁守護到最後】
  • 藉由【創造(想像)者】或的成就感，滿足【想加油添醋】的心情

BL作為女性想法的反射

  1. 男人是這樣想的吧 (感情上)
  2. 自己喜歡被這樣對待 (床上)

用愛情觀來解釋男人的【友情】

  • 明明感情很差，卻又互相認同對方
  • 想法明明完全和不來，但經常一起喝酒
  • 在學校中是競爭對手，總是很在意對方，感情不睦，可是眼裡總是看著對方

因為不知道要怎麼解釋這種關係，所以用愛情關係來解釋

還有女性都很喜歡戀愛話題

萌 的定義 (叫少女心比較貼切)

【雖然沒有色情要素，但我絕對全面支持 OOO ，不管誰怎麼想，為了她的幸福我什麼都願意做】

in my opinion, 這個定義已經很接近一般的少女心了

而這個定義有解釋到【不懂規則又有什麼關係，看他們那麼努力就夠了】

【王牌投手】是一部有指標性的作品

從少女心到母性: 兩者的差異

加油 (好壞都可，有努力就好) -> 希望他好好的

萌到BL的連續過程

在一個【自己不存在的世界】，讓【可愛的存在】療癒自己

故在此 可愛 的定義很重要

【可愛的萌角色】->【男孩子氣的女生】->【偽娘】->【正太】->【可愛的男生】->【型男可愛的一面】->【有點崩壞的(老)男人脆弱的一面很可愛】->【連壯漢歐吉桑也覺得很可愛】

在日本 可愛 的定義

欣賞事物的缺點與弱點 => 與【留白】、【不完整性】、【省略】等都是一樣的道理

腐男子不是GAY

性別就是一種屬性，類似貓耳等

萌與性不同
也與戀愛不同

呼應萌與戀愛不同，觀察對象不是戀愛對象

你所堅持的萌點是?

  • 比朋友更深的關係，但又還不到戀人的地步，絕妙的距離感
  • 異性也好同性也好，希望聯繫彼此關係的是【獨一無二的理由】，無論是好是壞都喜歡
  • 眼鏡、保力、攻君面前的受君，低級的(嘴臭) => 想看不常見的
  • 共犯關係、無血緣關係的兄弟、老師與學生、模擬家人 => 隱密性強的關係、【朋友以上戀人以上家人未滿】
  • 享受角色闖入別人心中或揭開別人瘡疤的樂趣
  • 僧侶 => 反差感
  • 美國西岸的高中生活 => 幻想世界、另一個世界
  • 【名偵探x助手】的組合 => 技士主從關係，又存在著對彼此的信賴
  • 乍看之下狂妄任性的【大爺】角色，其實內心纖細敏感，比受君還少女心的攻君。進可攻，退可受的美味角色
  • 【只要有這人在就放心了】的信任感 => 安全感
  • 反差大的角色，看起來冷酷，卻似乎有許多孩子氣的地方 => 和不同對象在一起時展現令人意外的一面
  • 青梅竹馬 - 認識者麼久卻發現對方有自己不知道的事，從這裡感情又出現進展。明明在一起這麼久卻有新發現
  • 令人捉摸不定的攻x老實認真的受
  • 摯友、競爭對手、老夫老妻
  • 【攻超愛受】。攻最好是在一般人眼中【立場比較低】的人，這樣地位才會對等 => 逃避男女關係與社會階級
  • 兩人個性與思考正好相反的設定，即使彼此無法理解，卻又強烈嚮往對方。分不清是愛戀還是崇拜。 => 來自自卑的萌，故事在為對方克服自卑會受到對方肯定之中得到發展
