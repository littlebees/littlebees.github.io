local install

helm pull <repo>/<name>會拉tar回來，name-ver.tar

helm可以install by

  • tar
  • folder
    但要指定完整檔名，所以如果都拉最新版，tar的ver會無法確認

所以要先untar，他untar後只會有名字
helm pull <repo>/<name> --untar
之後就可以直接指定folder name安裝
