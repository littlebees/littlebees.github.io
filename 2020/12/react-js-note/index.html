class & hook

class жҳҜеңЁеҗҢдёҖе…ғд»¶дёӢзҡ„жүҖжңүзӢҖж…Ӣе…ұз”ЁеҗҢдёҖеҖӢlife circle(жүҖд»ҘдёҖж—ҰеҫҲеӨҡдәӢиҰҒеҒҡпјҢжңғеҫҲж“ )пјҢ

з„¶иҖҢhookжңғиҮӘе·ұйҷ„еҲ°зӣёе°ҚжҮүзҡ„life circleйҡҺж®өпјҢеҰӮжһңstateд№Ӣй–“дә’еғҸеҪұйҹҝжңғи®“codeи·ідҫҶи·іеҺ»зҡ„пјҢи®Ҡеҫ—дёҚеҘҪзңӢ

ж•ҙй«”иҖҢиЁҖ(иіҮж–ҷзҡ„еӮійҒһж–№ејҸ)

дёҖи·Ҝеҗ‘дёӢпјҢиіҮж–ҷйғҪжҳҜеҫһparentеҲ°childпјҢйғҪз”ЁattrеӮіеҲ°propsдёӯпјҢchildиҰҒж”№и®ҠдёҠеұӨзҡ„зӢҖж…ӢеҸӘиғҪйҖҸйҒҺ

  1. дҫҶиҮӘparentзҡ„callbackдҫҶж”№и®ҠзӢҖж…Ӣ
  2. contextдҫҶеӮійҒһ

дҪҶйҖҷжЁЈеҫҲverboseпјҢд№ҹжңғеҫҲж··дәӮпјҢдёҚеҘҪtraceпјҢйғҪжҳҜеӣ зӮәиіҮж–ҷеҸӘиғҪдёҖи·Ҝеҗ‘дёӢпјҢе°ҚжӯӨйҮҚж§Ӣзҡ„ж–№ејҸжңү

  1. еҫһжәҗй ӯи§Јжұә: жҠҠstateеҫҖдёҠжҠҪгҖҒhigh-order-componentгҖҒpropsвҖҷ render function
  2. жҠҪеҲ°е…Ёеҹҹ: flux

еҸҜжҺ§иҲҮдёҚеҸҜжҺ§е…ғд»¶

иіҮж–ҷзҡ„и·Ҝеҫ‘йғҪжҳҜ

  1. eventжңғзў°еҲ°дҫҶиҮӘparentзҡ„callback
  2. callbackжңғsetstateиҮӘе·ұ(parent)зҡ„state
  3. жңҖеҫҢйҖҸйҒҺpropsеҺ»render

дҪҶжҳҜпјҢе…©еҖӢе·®еңЁиіҮж–ҷзҡ„дҫҶжәҗдёҚеҗҢ

еҸҜжҺ§

  1. eventжңғзў°еҲ°дҫҶиҮӘparentзҡ„callback (inputеҫһeventдҫҶ)
  2. callbackжңғsetstateиҮӘе·ұ(parent)зҡ„state
  3. жңҖеҫҢйҖҸйҒҺpropsеҺ»render

дёҚеҸҜжҺ§

  1. eventжңғзў°еҲ°дҫҶиҮӘparentзҡ„callback
  2. callbackжңғsetstateиҮӘе·ұ(parent)зҡ„state (inputеҫһrefдҫҶ)
  3. жңҖеҫҢйҖҸйҒҺpropsеҺ»render

еӣ зӮәrefиў«з”ЁдҫҶз¶ҒdomпјҢжүҖд»ҘеҰӮжһңжҳҜиҰҒеҫһеҫҲдёҠеұӨеӮіrefдёӢеҺ»жңғеҫҲз—ӣиӢҰпјҢжүҖд»ҘжңүforwardrefпјҢе…¶еҜҰе°ұжҳҜз”ЁfunctionеҺ»жҺҘrefпјҢиҖҢдёҚжҳҜз”Ёprops

еӨ§жҰӮзҡ„жј”еҢ–жөҒзЁӢ

View + Controller(ViewModel, representation model) => View + Controller + Model => View + Route + Controller + Model

дёҖй–Ӣе§ӢviewиҲҮйӮҸијҜжҳҜе’ҢеңЁдёҖиө·пјҢеҫҢйқўзӮәдәҶз®ЎзҗҶиЁұеӨҡиіҮж–ҷпјҢе…ҲжҠҠviewиҲҮйӮҸијҜеҲҶеҮәдҫҶпјҢи®ҠжҲҗview(presentation component)иҲҮcontroller(container component)пјҢжңҖеҫҢжҠҠзӢҖж…ӢжҠҪеҲ°еҗҢдёҖеҖӢең°ж–№пјҢжҲҗзӮәmodel

Flux

е°ұжҳҜжңүйҷҗзӢҖж…Ӣж©ҹпјҢеңЁMVCдёӯзҡ„model

еӨ§жҰӮе°ұжҳҜ

  1. жҠҠжүҖжңүйӮҸијҜж”ҫеҲ°зӢҖж…Ӣж©ҹдёӯ(dispatcher, reducer)
  2. жҠҠзӢҖж…ӢиҲҮиіҮж–ҷж”ҫеңЁе…ЁеҹҹдёӯпјҢйҖҷжЁЈеӨ§е®¶йғҪзңӢзҡ„еҲ°пјҢдёҚдҪҝз”ЁcallbackжҲ–жҳҜref(store)
  3. з”Ёsubscribeзҡ„ж–№ејҸдҫҶи®“viewзҹҘйҒ“storeжӣҙж–°

йҖҷиЈЎе°ұжҜ”ијғиІјиҝ‘еҺҹжң¬зҡ„MVCпјҢж•ҙеҖӢжөҒеӢ•жҳҜдёҖеҖӢдёүи§’еҪўпјҢ
иҖҢwebзҡ„MVC(model2)пјҢе°ұжҳҜдёҖеҖӢеҗ‘дёҠзҡ„е°–е°–пјҢmodelиҲҮviewд№Ӣй–“зҡ„жәқйҖҡйғҪиҰҒйҖҸйҒҺcontrollerе®ҢжҲҗ

Redux

еҹәжң¬дёҠе°ұжҳҜFluxпјҢдҪҶ

  1. еҸӘжңүдёҖеҖӢstoreпјҢжүҖд»ҘиҰҒжӢҝеҲ°зӢҖж…ӢиҲҮactionпјҢиҰҒеҺ»еҲҮеҮәйҒ©еҗҲзҡ„йғЁд»ҪзөҰcomponentпјҢжІ’иҫҰжі•зӣҙжҺҘsubscribeпјҢжүҖд»ҘжңүжүҖи¬Ӯзҡ„connect
  2. stateжҳҜзӣҙжҺҘж”ҫеңЁеҸғж•ё(еҰӮжһңеҶҚж”№дёҖдёӢпјҢе°ұеҸҜд»Ҙз”Ёstate monadеҺ»еҢ…ж•ҙеҖӢreducer)пјҢдёҚеғҸе…¶д»–жҳҜзӣҙжҺҘж”№е…Ёеҹҹи®Ҡж•ё
