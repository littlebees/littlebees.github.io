css的z-index與position

Posted on 2020-05-04 Edited on 2022-07-30 In Web , CSS , FAQ

z-index是怎麼來的

東西疊上去就會有z軸

要疊上去一定要地基(stacking context，如果你愛術語的話)，那地基如何產生?

地基如何產生?

  • 根元素（<html></html>）
  • position 設定成 fixed 或 sticky
  • position 設定成 relative 或 absolute & 有設定 z-index
  • 有設定 transform 屬性
  • opacity 的值設定小於 1

如果有許多地基，要怎麼辦?

地基的順序

position & translate

position可以用top之類的來調整位置
那他與translate不同在哪?

用translate的話，會產生地基，但如果用position卻沒有加上z-index，
就會與其他原本的居民在一起，到時候如果有top之類的變動，會牽動其他居民一起來(reflow)

所以要用position與top之類的要記得加上z-index，或是用translate
