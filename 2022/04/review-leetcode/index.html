48. Rotate Image

有兩個解法

一個是4個一起換

一個是對角線 (固定走橫的，想像成轉成直線)
(i, j), (n-1-i, n-1-j)
另一個反對角線 (固定走直的，想像成轉成橫線)
(n-1-j,i), (j,n-1-i)
之後把覆蓋順序寫下來
(i, j) <- (n-1-j, i) <- (n-1-i, n-1-j) <- (j, n-1-i)

另一個是transpose(對角線互換)後把column都reverse

287. Find the Duplicate Number

要找什麼?
找總量不對的第一個位置

b帶nums.size()-1，因為最後出來的最大數字就是這個值
需要捨棄的部分帶a = mid+1
要保留的部分帶b = mid
return就是a
