assm

aasm就是狀態機，這邊說說設計上的注意點

  1. callback只有after與guard可以傳參數
  2. 參數最好直接傳hash
  3. 所有callback都要抽成private method，之後測試才能stub
  4. 注意: 當after失敗時，他不會自動rollback狀態
