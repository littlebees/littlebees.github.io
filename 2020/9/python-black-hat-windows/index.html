CH8

主要是讓大家看看ctypes與pyhook的操作

ctypes is a foreign function library for Python.
It provides C compatible data types, and allows calling functions in DLLs or shared libraries.
It can be used to wrap these libraries in pure Python.

The pyHook package provides callbacks for global mouse and keyboard events in Windows.
Python applications register event handlers for user input events such as left mouse down, left mouse up, key down, etc. and set the keyboard and/or mouse hook

keylogger.py

就是掛callback這沒問題
但要注意的是handler
可以看成windows programming的參考(或是ptr)

screenshotter.py

Device Contexts(DC)就是呈現(或放置)圖像的地方
這code有三個DC

  1. desktop
  2. memory(放圖的地方)
  3. bitmap(圖檔)

詳細的Device Contexts要搭配GDI programming來看

整個流程是
desktop -> memory -> bitmap -> .bmp

sandbox_detect.py

所以按鍵次數相對固定

有趣的是這裡處理按鍵是用ctypes硬幹，其實也可以用pyhooks

CH10

這裡的目的是找出

高權限process會使用的檔案，但其檔案可以被低權限的使用者取用的檔案

以此繞過權限管制

下面的兩個code就是分別

  1. 找出高權限(有趣的)process
  2. 當檔案被修改時插入我們的code

process_monitor.py

把所有新產生的process的權限列出來

這邊有一些有趣的權限，但就不列出詳細的作用(懶)

  • SeBackupPrivilege
  • SeDebugPrivilege
  • SeLoadDriver
