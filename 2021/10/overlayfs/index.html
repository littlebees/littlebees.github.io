動機#

記錄用

overlayfs#

就是疊疊樂

lower提供基底(通常是readonly或是另一個overlay)，upper提供可以寫的環境 overlay就是兩個疊在一起結果

用法 mount -t overlay overlay -o lowerdir=/lower,upperdir=/upper,workdir=/work /merged * workdir: 就是暫存，要是空的folder

readonly overlay#

我只要把多個folder疊再一起怎麼辦?

mount -t overlay overlay -o lowerdir=/lower1:/lower2 /merged

用來疊客製化#

mount -t overlay overlay -o lowerdir=/mnt/sq,upperdir=/x/pers/rw,workdir=/x/pers/work /mnt/merged mksquashfs /mnt/merged /...
